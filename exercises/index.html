Promises (1/2)

  1. add number property to Account class
  2. add findAccount(accountNumber) method to Person class (use Array.prototype.find)
  3. add withdraw(accountNumber, amount) method to Person class
  4. withdraw method returns a promise (create promise with new Promise(resolve, reject) and return it)
  5. promise is resolved after 3 seconds (use setTimeout(callback)) when the account is found (use findAccount method) and the given amount is higher than foundAccount.balance

Promises (2/2)

  1. when promise is resolved, the account balance is updated and success message appears (with account number, new balance, amount of withdrawn money)
  2. promise is rejected either when the account can't be found or when the requested amount is bigger than account balance
  3. in case of rejection, there is an appropriate message returned
  4. update all accounts creation (Account constructor calls) with account numbers
  5. withdraw some money for different cases (use person.withdraw(accNumber, amount).then(...).catch(...)
